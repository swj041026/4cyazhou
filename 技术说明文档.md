# 崖城风骨——崖州古城建筑文化数字图鉴 技术说明

## 一、项目概述

**项目名称**: 崖城风骨——崖州古城建筑文化数字图鉴

**项目性质**: 全国大学生计算机设计大赛参赛作品（信息可视化·交互信息设计赛道）

**主题内容**: 崖州古城建筑文化（聚焦1911年以前的历史文化）

**叙事结构**: 采用「从物到魂」四层递进结构：
- **形·古城** - 古城形制与布局
- **技·营造** - 建筑技艺与工艺
- **人·人文** - 历史人物与人文脉络
- **魂·传承** - 文化精神与传承

## 二、前端技术栈

### 2.1 基础技术

| 技术 | 版本/说明 | 用途 |
|------|-----------|------|
| **HTML5** | 标准规范 | 页面结构与语义化标签 |
| **CSS3** | 现代浏览器支持 | 样式设计、动画、响应式布局 |
| **JavaScript (ES6+)** | ES6+ 语法 | 交互逻辑与动态效果 |

### 2.2 字体资源

项目使用 Google Fonts 提供的开源中文字体：

- **Noto Serif SC**（思源宋体）- 用于标题和正文，营造古典文化氛围
- **Noto Sans SC**（思源黑体）- 用于界面文字，保证可读性

引用方式：
```html
<link href="https://fonts.googleapis.com/css2?family=Noto+Serif+SC:wght@400;600;700&family=Noto+Sans+SC:wght@300;400;500;700&display=swap" rel="stylesheet">
```

### 2.3 前端功能特性

#### 视觉与动画
- **CSS 动画**: 滚动显示动画(scroll-reveal)、悬浮效果、过渡动画
- **SVG 图标**: 内联 SVG 实现各种图标（导航、按钮、装饰元素）
- **渐变与阴影**: 使用 CSS3 渐变和阴影增强视觉层次

#### 交互组件
- 智能助手浮动窗口（支持拖拽、调整大小）
- 模态弹窗系统
- 响应式导航菜单
- 背景音乐控制
- 字体大小切换
- 回到顶部按钮
- 页面进度条

#### 响应式设计
- 适配桌面端、平板和移动设备
- 使用媒体查询实现断点切换

## 三、后端技术栈

### 3.1 服务器环境

| 技术 | 说明 |
|------|------|
| **Node.js** | JavaScript 运行时环境 |
| **原生 http 模块** | 创建 HTTP 服务器 |
| **原生 https 模块** | 用于 API 请求转发 |
| **fs 模块** | 文件系统操作 |
| **path 模块** | 路径处理 |

### 3.2 AI 智能助手服务

- **API 提供商**: DeepSeek
- **模型**: `deepseek-chat`
- **功能**: 崖州古城知识问答
- **系统提示词**: 包含崖州古城的历史背景、建筑文化等专业知识

### 3.3 服务器脚本

项目提供两个服务器脚本：

1. **基础服务器** (`server.js`)
   - 仅提供静态文件服务
   - 适合纯展示场景

2. **AI 服务器** (`api-server.js`)
   - 提供静态文件服务
   - 集成 DeepSeek AI 聊天接口
   - 支持健康检查接口

## 五、运行方式

### 5.1 环境要求

- Node.js 环境（建议 v14+）
- 现代浏览器（Chrome、Firefox、Edge、Safari）

### 5.2 启动方式

#### 方式一：基础服务器（仅静态文件）

```bash
node server.js
```

访问地址：`http://localhost:8080`

#### 方式二：AI 服务器（含智能助手功能）

```bash
node api-server.js
```

访问地址：`http://localhost:8080`

**注意**: 使用 AI 功能需要在 `.env` 文件中配置 DeepSeek API 密钥。

### 5.3 环境变量配置

在项目根目录创建 `.env` 文件：

```env
DEEPSEEK_API_KEY=your_api_key_here
DEEPSEEK_BASE_URL=https://api.deepseek.com
```

## 六、核心功能说明

### 6.1 AI 智能助手 "崖城智查"

- **位置**: 页面右下角浮动窗口
- **功能**: 
  - 崖州古城历史问答
  - 建筑文化知识查询
  - 历史人物介绍
- **交互**: 支持拖拽移动、调整大小、最小化
- **快捷问题**: 提供预设常见问题快捷按钮

### 6.2 四层叙事结构

| 层级 | 页面 | 内容重点 |
|------|------|----------|
| 形 | page-shape.html | 古城形制、城门布局、街巷结构 |
| 技 | page-technology.html | 建筑工艺、材料技术、营造法式 |
| 人 | page-people.html | 历史人物、人文故事、文化传承 |
| 魂 | page-spirit.html | 文化精神、传承价值、现代意义 |

### 6.3 交互特性

- **滚动动画**: 元素随滚动渐显
- **悬浮效果**: 卡片悬浮时放大、阴影变化
- **轮盘交互**: 历史人物轮盘选择
- **时间线**: 历史事件时间线展示
- **模态弹窗**: 详情内容弹窗展示
